# Kavin Balaji S - Personal Portfolio

A modern, responsive personal portfolio website built with React, Tailwind CSS, and Framer Motion. Features sleek dark theme with particle animations and interactive elements.

## Features

- **Modern Design**: Clean, dark-themed UI with smooth animations
- **Responsive**: Optimized for mobile, tablet, and desktop devices
- **Particle Background**: Subtle animated particle network effect
- **Interactive Animations**: Framer Motion powered animations and micro-interactions
- **Typing Animation**: Dynamic role titles with typewriter effect
- **Contact Form**: Functional contact form with EmailJS integration
- **Certifications**: Showcase professional certifications with verification links
- **Performance Optimized**: Fast loading with Vite bundling

## Tech Stack

- **Frontend**: React 18, JavaScript (JSX)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: React Icons (HeroIcons, Simple Icons), Lucide React
- **Form Handling**: EmailJS
- **Scrolling**: React Scroll
- **Build Tool**: Vite
- **Fonts**: Inter (Google Fonts)
